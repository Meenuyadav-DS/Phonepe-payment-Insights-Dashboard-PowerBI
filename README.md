📊 PhonePe Payment Insights Dashboard (Power BI)
📌 Project Overview

This project is an interactive Power BI dashboard built to analyze PhonePe payment transactions. It provides insights into transaction volume, transaction value, user demographics, payment status, monthly trends, and weekend vs weekday usage.

🎯 Objectives
1.Analyze total transactions and transaction value
2.Monitor payment success rate
3.Track monthly transaction trends
4.Analyze age-wise user distribution
5.Compare weekday vs weekend usage
6.Identify top users based on transaction value
7.Filter data using interactive slicers

📈 Dashboard Features
1.KPI Cards
  -Total Transactions
  -Total Transaction Value
  -Unique Users
  -Successful Transaction Rate
2.Monthly Transaction Trend
3.Age Segment Distribution
4.Payment Status Analysis
5.Top 5 Users
6.Weekend vs Weekday Usage
7.Service-wise Transaction Analysis
8.Interactive Month Filters
